# Phase 3 test suite — glow / calibrated confidence

Goal: the glow layer produces a **calibrated** confidence that beats a no-glow model on ECE **without**
buying it by lowering accuracy, and the frequency-runaway fix is provably load-bearing.

**Precondition:** Phase 2 (memory) done. **Headline gate:** H5 (see
[`../METRICS-AND-GATES.md`](../METRICS-AND-GATES.md) §5).

---

## Cards

### T3.1 — Brightness + margin, Born-consistent
- Files: `qilm-core/src/glow.rs`, `qilm-oracle/tests/kat_glow.rs`
- Signature: `pi_c = normalize(log(1+n_c)·idf(c))`; `conf = pi_top1 − pi_top2`; `|α_c| = sqrt(pi_c)`.
- Done-when: `kat_glow_born` GREEN — `p(c) = |α_c|² == pi_c` within `1e-6` (Born consistency).

### T3.2 — ECE, pinned bins + accuracy guard
- Files: `qilm-train/src/metrics/ece.rs`, `qilm-train/tests/nc_glow.rs`
- Signature: `ece(confidences, correct, bins=15)`.
- Done-when: `test_ece_known` GREEN — on a synthetic confidence/label set with a **hand-computed** ECE,
  the function returns it within `1e-6` (expected value in the test comment).
- Stays-red:
  - `test_ece_bins` — calling ECE with `M=1` must be rejected/asserted (a 1-bin ECE is meaningless);
    the mutant "bins=1" is killed here.
  - `nc_glow_accuracy_guard` — a `uniform_confidence_glow()` variant that lowers all confidences must
    **FAIL** H5 (the accuracy guard trips even though raw ECE dropped). *(Threat T4: calibration bought
    by dumbing down.)*

### T3.3 — Runaway guard is load-bearing
- Files: `qilm-train/tests/nc_runaway.rs`
- Done-when: `nc_runaway_off` GREEN — with the three fixes (log-compress / normalize / IDF) **off**,
  the brightest concept is the highest-frequency byte (space/'the'); with them **on**, it is not.
  Asserts the fix is causal, not cosmetic.

### T3.4 — Robustness rises with exposure
- Done-when: `test_robustness_monotone` GREEN — stability hit-rate (Phase 2) is non-decreasing in the
  exposure count `n_c` across the pre-registered exposure schedule.

### T3.5 — Run H5, emit via report
- Done-when: `results/RESULTS.md` shows, from provenance-checked runs over ≥5 seeds,
  `ECE(glow) ≤ ECE(no-glow) − 0.02 ∧ acc(glow) ≥ acc(no-glow) − 0.5pt ∧ p<0.05` → **PASS**; else
  **NARROW to salience** (report measured ΔECE and Δacc).
