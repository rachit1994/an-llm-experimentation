# Phase 4 test suite — complex/phase and the decisive H1 ablation

Goal: decide whether **phase** (the wave mechanism) earns its parameters, via a fair, param-matched,
equal-tuning `phase=0` ablation. This is the branding-decisive test; the whole point is that it can
return "no."

**Precondition:** the harness (Phase 0) is trusted; a working objective (Phase 1). **Headline gate:**
H1 (see [`../METRICS-AND-GATES.md`](../METRICS-AND-GATES.md) §6).

---

## Cards

### T4.1 — The `phase=0` switch is a true ablation (param-matched)
- Files: `qilm-core/src/embedding.rs`, `qilm-train/tests/test_phase_ablation.rs`
- Signature: `phase=0` zeroes the imaginary channel **without removing parameters** (so param counts
  match — otherwise you confound phase with capacity).
- Done-when: `nc_param_match` GREEN between full-complex and `phase=0` (within ±5%).
- Kills-mutant: implement `phase=0` by deleting the imaginary tables (param mismatch) → `nc_param_match`
  red. *(This mutant is the classic rigged ablation; the control must catch it.)*

### T4.2 — Phase-usage diagnostic
- Files: `qilm-train/src/metrics/phase_hist.rs`
- Done-when: `test_phase_hist` GREEN — for a model with deliberately non-trivial phases, the histogram
  is non-degenerate; for `phase=0`, it is a spike at 0. If the *free* model learns `φ→0` on its own,
  that is an H1 kill (the data did not want phase) — asserted in T4.3.

### T4.3 — Run H1: effect size AND significance
- Procedure: on Cut A (AG News) and Cut B (byte-LM), full-complex vs `phase=0`, equal tuning, ≥5 seeds.
- Done-when: `results/RESULTS.md` shows, from provenance-checked runs, the paired decision:
  ```
  Δ = metric(complex) − metric(phase=0)
  PASS (keep phase) ⟺ Δ ≥ δ_phase (1.0 acc pt / 2% rel BPB) AND paired p<0.05 AND phase used
  else              ⟹ DROP the phase/"wave" claim  (report Δ, p, and the phase histogram)
  ```
- Anti-fake reminder: a `phase=0` tie is a **valid, publishable result** (drop the claim); do not tune
  the complex arm harder than the ablation to manufacture Δ — `equal tuning budget` is enforced by
  the config and checked in `verify-results`.

## Phase-4 DoD checklist

- [ ] T4.1 ablation is param-matched; the delete-imaginary mutant is killed by `nc_param_match`.
- [ ] T4.2 phase histogram behaves; the `φ→0` self-collapse case is detected.
- [ ] T4.3 H1 verdict generated by `report` with both effect size and paired significance over ≥5
      seeds, on the pre-registered `δ_phase`.
