# Phase 6 test suite — generation + readout mitigations (the O(|V|·d) wall)

Goal: measure how far Born generation is from parity, and whether a readout mitigation
(low-rank / tied / hierarchical / VQ) reaches acceptable quality at sub-`O(|V|·d)` cost. The honest
default outcome is **narrow to encoder-only**; the tests must make that a measured verdict, not a vibe.

**Precondition:** Phases 1 (objective) + 2 (memory). **Headline gate:** a mitigation on the
quality-vs-cost frontier (see [`../PHASE-6-generation-multimodal.md`](../PHASE-6-generation-multimodal.md)).

---

## Cards

### T6.1 — Readout cost is measured, not assumed
- Files: `qilm-train/src/metrics/readout_cost.rs`, `qilm-train/tests/kat_readout_cost.rs`
- Done-when: `kat_readout_cost` GREEN — measured MAC/step and param counts for {full, low-rank r=64,
  VQ K=1024, hierarchical, tied} match the closed-form values in
  [`../METRICS-AND-GATES.md`](../METRICS-AND-GATES.md) / white-paper §8.2 within `1%`
  (full=16.38M, low-rank=2.08M, VQ=0.52M, hier≈7.66k). *(Catches a mitigation that claims a saving it
  does not actually deliver.)*

### T6.2 — Generation quality on the synthetic source
- Files: `qilm-train/tests/kae_generation.rs`
- Done-when: `kae_generation` GREEN — each mitigation's generation BPB on `synth_markov` is reported vs
  the full-Born BPB and vs the source entropy `H`; the frontier point (BPB, cost) is emitted. A
  mitigation "passes" only if `BPB(mitigation) ≤ BPB(full) + 0.05` at strictly lower cost.

### T6.3 — Coconut-style feedback amortization
- Done-when: `test_pattern_feedback` GREEN — feeding the continuous pattern back (no per-step decode)
  produces the same distribution as per-step decode at emission points, within tolerance, at lower
  measured readout cost.

### T6.4 — Multimodal is a toy, labeled as such
- Done-when: `test_shared_space_toy` GREEN on a **small synthetic** two-modality set with from-scratch
  encoders (C1) — the shared-space *mechanism* works; the report explicitly labels this a mechanism
  demonstration, **not** a quality claim (guards against overclaiming cross-modal results, per
  white-paper §10).

## Phase-6 DoD checklist

- [ ] T6.1 readout costs measured within 1% of the closed form.
- [ ] T6.2 each mitigation's (BPB, cost) frontier point generated by `report`.
- [ ] T6.3 pattern-feedback equivalence verified.
- [ ] T6.4 multimodal toy runs from scratch and is labeled a mechanism demo.
- [ ] Verdict: either a named mitigation reaches parity sub-`O(|V|·d)` (**PASS**), or **narrow to
      encoder-only** with the measured frontier as evidence — never an unmeasured "it works."
